# Лабораторная работа №6: Продвинутый HTML. Введение в CSS стили. Создание простого сайта про историю ЭВМ

**ФИО:** Федотова В.С. 
**Группа:** ИСП-232
___

## Цель работы

Освоить базовые и продвинутые возможности HTML и CSS, создать семантически верный, стилизованный веб-сайт на тему "История ЭВМ" с использованием современных технологий вёрстки.

## Задачи работы
- Изучить разницу между блочными и строчными элементами.
- Освоить работу с тегами форматирования текста.
- Разобраться с типами ссылок: относительные, абсолютные, якорные, mailto, tel.
- Научиться добавлять изображения с указанием путей и атрибутами доступности.
- Использовать теги для цитат (blockquote, cite).
- Встраивать мультимедиа (видео, аудио, iframe).
- Создавать формы с различными типами полей ввода.
- Использовать семантические теги HTML5 (header, nav, main, section, aside, footer).
- Применять базовые стили CSS для оформления страниц.
- Освоить работу с Git и GitHub.
