# Лабораторная работа №7: Адаптивная верстка с использованием Flexbox
___

## Цель работы

Освоить современные технологии CSS-разметки — Flexbox, научиться создавать адаптивные веб-страницы, которые корректно отображаются на устройствах с разными размерами экранов.

## Задачи работы
- Изучить базовые концепции Flexbox.
- Настроить рабочую среду в VS Code.
- Создать базовую HTML-структуру и подключить CSS-стили.
- Практиковаться с основными свойствами Flexbox:
    - flex-direction
    - justify-content
    - align-items
    - flex-wrap
    - gap
    - flex-grow, flex-shrink, flex-basis
    - align-self
- Разработать практические примеры:
    - Навигационное меню
    - Карточки товаров
    - Центрирование элемента
- Научиться работать с Git и GitHub.
- Оформить результаты работы в файле README.md.
